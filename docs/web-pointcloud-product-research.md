# Web 三维点云产品功能调研与迁移清单

## 调研对象与可借鉴方向

- Potree / CesiumJS 类 Web 点云查看器：强调大规模点云浏览、EDL 轮廓增强、点大小/点衰减、裁剪、测量、视图定位与性能控制。
- Pix4Dcloud：强调在线 2D/3D 对比、测量、标注、虚拟巡检、进度记录和分享协作。
- NavVis IVION：强调浏览器内访问现实捕获数据、点云/全景/模型对比、资产标记、问题协作、面向非专业用户的清晰导航。
- Trimble Connect：强调 2D/3D Viewer、评论、标签、文件共享、对象组织和项目协作。

## 对当前项目的启发

当前 viewer 已具备 LAS 本地加载、RGB/高程/类别显示、类别颜色、裁剪、连续测距、EDL、航线/部件点/视锥/相机预览、航线编辑等基础能力。短期最值得迁移的是“低耦合、高感知”的浏览体验功能，而不是账户、云协作、3D Tiles 服务端切片这类重型能力。

## 优先迁移清单

1. 视角导航小组件
   - 价值：提升空间方向感，接近成熟 3D Web Viewer 的基本体验。
   - 可移植性：高，只依赖当前相机 yaw/pitch。
   - 第一版：右上角显示 X/Y/Z 方向轴，并提供俯视、前视、侧视、等轴测快捷按钮。

2. 点云点大小衰减
   - 价值：近处更清晰、远处更收敛，浏览输电通道时视觉层次更好。
   - 可移植性：高，只需 shader 增加开关。
   - 第一版：提供“近大远小”开关，不改变原始点坐标与分类。

3. 截图导出
   - 价值：巡检沟通、问题记录和阶段汇报常用。
   - 可移植性：高，直接从 viewer canvas 导出 PNG。
   - 第一版：工具栏增加截图按钮，导出当前 WebGL 画面。

4. 视图书签 / 标注点
   - 价值：对应 Pix4D/NavVis 的标注与问题定位。
   - 可移植性：中，需要保存相机状态和文本。
   - 后续：支持添加问题点、备注、导出 JSON。

5. 截面/剖面视图
   - 价值：对应 Potree 常用 Profile 功能，适合电力通道净空查看。
   - 可移植性：中，需要点选折线和局部投影统计。
   - 后续：在测量工具基础上扩展。

6. 点云质量/性能面板
   - 价值：类似 Cesium 的缓存/LOD 观测，帮助解释浏览卡顿。
   - 可移植性：高。
   - 后续：显示渲染点数、抽样比例、GPU/WebGL 能力和帧耗时。

## 当前实现批次

第一批落地：视角导航小组件、快捷视角、截图导出、点大小衰减。

第二批落地：视图书签、问题点标注、标注详情编辑、标注 JSON 导出。该批次对应 Pix4Dcloud / NavVis / Trimble Connect 等产品常见的在线标注、资产/问题记录、视图复盘能力。

第三批落地：剖面/Profile 分析。用户在点云中选择两点后，系统按可调通道宽度抽取当前渲染点集，生成沿线距离-高程散点剖面，并在主视图叠加剖面线。该能力对应 Potree 等点云查看器常见的 profile 工具，适合输电走廊地形、导线、杆塔附近高度分布的快速核查。

第四批落地：净空辅助分析。基于当前抽样点云分类，识别导线/地线类点和地面、植被、建筑、交跨等地物类点，按可调水平搜索半径估算垂直净空，列出低于阈值的风险点并在主视图中高亮。该能力是电力巡检场景相较通用点云查看器更有业务价值的迁移方向，第一版采用浏览器端近似计算，后续可扩展为线路分段、导线拟合和报告导出。
